Name

holorc - configuration file for holo(8)

Description

The holorc file defines which plugins will be loaded and used by Holo, and in which order. Blank lines, and comment lines starting with a # character are ignored.

The format of each command line is:

plugin $PLUGIN_ID

where $PLUGIN_ID is the alphanumeric identifier of the plugin. The plugin identifier is encoded in several paths that are relevant for the plugin:

  • The plugin executable is installed at /usr/lib/holo/holo-$PLUGIN_ID.

  • Configuration packages install resources for this plugin below /usr/share/holo/$PLUGIN_ID/.

  • The plugin stores its state at /var/lib/holo/$PLUGIN_ID/.

Best practices

Plugins are encouraged to add themselves to /etc/holorc at install time by installing a holoscript, as detailed in the holo-plugin-interface(7) specification. If plugins are installed this way, you can observe this with the holo scan command. For example:

$ holo scan /etc/holorc

/etc/holorc
    store at /var/lib/holo/files/base/etc/holorc
    passthru /usr/share/holo/files/00-holo-users-groups/etc/holorc.holoscript
    passthru /usr/share/holo/files/95-holo-run-scripts/etc/holorc.holoscript

See also

holo(8), holo-plugin-interface(7)
